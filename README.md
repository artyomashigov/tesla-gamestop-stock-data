# Tesla and GameStop Stock Data

Notebook project analyzing Tesla and GameStop stock and revenue data.

## Project Goal

The project collects and visualizes market data for Tesla and GameStop. It combines stock-price history with revenue data to show how each company changed over time.

## Files

- `TeslaGamestop.ipynb` - full notebook with data collection, cleaning, and visualizations.

## Workflow

1. Collect stock data for Tesla and GameStop.
2. Collect or scrape revenue data.
3. Clean and align the datasets.
4. Build visualizations for stock price and revenue trends.
5. Compare the two companies through charts.

## Tools

Python, pandas, yfinance/web scraping tools, and Jupyter Notebook.
